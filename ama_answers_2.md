**1. What are Internal CSS and External CSS?**

* **Internal CSS:** Written inside the `<style>` tag in the `<head>` section of an HTML page. Used for a single page.
* **External CSS:** Written in a separate `.css` file and linked to HTML using `<link>`. Used for multiple pages.

---

**2. What is the Interface Segregation Principle (ISP) in SOLID?**

It states that a class should not be forced to implement methods it does not use. Use smaller, specific interfaces instead of one large interface.

---

**3. What is the Anchor Tag in HTML?**

The `<a>` tag is used to create hyperlinks to navigate to another webpage.
Example: `<a href="url">Link</a>`

---

**4. What is the Image Tag in HTML?**

The `<img>` tag is used to display images on a webpage.
Example: `<img src="image.jpg" alt="image">`

---

**5. What are Attributes of the `<img>` Tag?**

Common attributes:

* `src` – image path
* `alt` – alternative text
* `width` – image width
* `height` – image height

---

**6. What is the `<strong>` Tag in HTML?**

The `<strong>` tag is used to show important text, usually displayed in **bold**.

---

**7. What are Primary Key and Unique Key in SQL?**

* **Primary Key:** Uniquely identifies each row and cannot contain NULL values.
* **Unique Key:** Ensures all values are unique but may allow NULL.

---

**8. What is the `<hr>` Tag in HTML?**

The `<hr>` tag creates a horizontal line to separate content.

---

**9. What is the `<br>` Tag in HTML?**

The `<br>` tag is used to insert a line break in text.

---

**10. What is IF ELSE in SQL?**

It is used for conditional execution of SQL statements based on a condition.

---

**11. How to Create an Image in HTML?**

Use the `<img>` tag with the `src` attribute.
Example: `<img src="photo.jpg">`

---

**12. What is a Candidate Key in SQL?**

A column or set of columns that can uniquely identify each row in a table.

---

**13. What is an Element Selector in CSS?**

It selects HTML elements by their tag name.
Example: `p { color: red; }`

---

**14. What are the Elements of the CSS Box Model?**

* Content
* Padding
* Border
* Margin

---

**15. Difference Between Class and ID Selector**

* **Class (`.`):** Can be used for multiple elements.
* **ID (`#`):** Used for one unique element.

---

**16. What is HTML?**

HTML (HyperText Markup Language) is the standard language used to create and structure webpages.
